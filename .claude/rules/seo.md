# SEO Configuration

- Site-level SEO params in `services/www/hugo.yaml` under `params`: `description`, `author`, `keywords`
- All blog posts should include a `description` field in front matter (under 160 characters)
- `robots.txt` is generated by Hugo from `services/www/layouts/robots.txt` template (`enableRobotsTXT: true`)
- `llms.txt` is a static file at `services/www/static/llms.txt` (llmstxt.org spec); must be manually updated when posts are added or removed
